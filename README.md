CONTENT
-------
Reports
- Run Reports
- Calculations
- Calculations Reports
- Saved Reports

INSTALL
-------
Copy the Reports/ folder and its content inside the modules/ folder of RosarioSIS.

Go to School Setup > School Configuration > Modules and click "Activate".

Requires RosarioSIS 2.8.25+
